## 📋 Recording Checklist

| Step | Action |
|---|---|
| ✅ Prepare browser | Open `ethack-delta.vercel.app` in Chrome, full-screen, no extensions visible |
| ✅ Screen resolution | 1920×1080 for crisp recording |
| ✅ Pre-load pages | Visit `/dashboard`, `/forecast`, `/enforcement`, `/field-officer`, `/advisory` before recording to avoid load spinners |
| ✅ Voiceover | Record voice separately, sync in post — speak at 120–130 words/min |
| ✅ Music | Subtle background — ambient/lo-fi, low volume |
| ✅ Cursor | Use a cursor highlighter tool (e.g., Loom, OBS) |
| ✅ Captions | Add English captions for accessibility |
